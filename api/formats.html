belso.formats.json_format.schema_to_json(schema, file_path=None)[source]

Convert a belso Schema to a standardized JSON format and optionally save to a file.

Args

  • schema (Type[belso.Schema]): the schema to convert.

  • file_path (Optional[Union[str, Path, PathLike]]): path to save the JSON to a file.

Returns

  • Dict[str, Any]: the converted schema.

belso.formats.json_format.json_to_schema(json_input)[source]

Convert a standardized JSON format or JSON file to a belso Schema.

Args

  • json_input (Union[Dict[str, Any], str]): either a JSON dictionary or a file path to a JSON file.

Returns

  • Type[belso.Schema]: the converted belso schema.
